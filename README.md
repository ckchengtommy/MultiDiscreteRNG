The MultiDiscreteRNG package provides functions for generating correlated count data with Generalized Poisson, Negative Binomial and Binomial Distribution marginal distributions. 


You can install the development version of MultiDiscreteRNG directly from GitHub using the devtools package:
<pre>

# install.packages("devtools")  # If you haven't installed devtools yet
devtools::install_github("ckchengtommy/MultiDiscreteRNG")

</pre>


Once the package is installed, simply load:

<pre>
library(MultiDiscreteRNG)
</pre>
